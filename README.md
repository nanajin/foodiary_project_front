# Foodiary
Typescript 기반 React 웹 프로젝트 'Foodiary'입니다!
- “오늘 뭐 먹지?” “오늘 뭐 먹었지?” 라는 분들께 추천드리는 나만의 다이어리 웹입니다.
- 본인의 취향에 따른 음식 추천과 일주일 식단 추천을 받을 수 있으며 먹은 음식에 대한 기록을 할 수 있습니다.
